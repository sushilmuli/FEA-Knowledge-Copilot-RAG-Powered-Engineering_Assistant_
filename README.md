# FEA-Knowledge-Copilot-RAG
• Developed an AI-powered chatbot using Retrieval
Augmented Generation (RAG) to answer queries from
FEA/LS-Dyna manuals/Regulation pdf.
• Implemented PDF ingestion using LangChain and
PyPDFLoader, Performed text chunking using Recursive
Character TextSplitter.
• Generated embeddings using OpenAI Embeddings and
stored in FAISS vector database,Built semantic search
and retrieval pipeline to fetch relevant engineering
content.
• Integrated LLM (GPT-4o-mini) to generate context
based accurate answers.
• Tools: Python, LangChain, FAISS, OpenAI API, Google
Colab
• Impact: Reduced manual search time for FEA
engineers and improved productivity

Note: To run this project, you'll need to obtain key files for OpenAI and NGROK
